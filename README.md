# Kate Hearts of Iron IV Syntax Highlighting 

Quick XML for [Kate Text Editor](https://kate-editor.org/) for Paradox Interactive's modding utility.

NOTE: Generated using Claude Opus 4.6 as a quick guide.

## Install

1. Clone this repository

2. Create the syntax folder if it does not already exist
**Linux/MacOS**
```sh 
cd kate-paradox-hoi4-syntax
mkdir -p ~/.local/share/org.kde.syntax-highlighting/syntax/
cp hoi4.xml ~/.local/share/org.kde.syntax-highlighting/syntax/
```

**Windows**

```plaintext
%USERPROFILE%\AppData\Local\org.kde.syntax-highlighting\syntax\
```

3. Restart Kate
4. Profit
